### Распределение CTR по группам

У тестовой группы больше пользователей с очень высоким и очень низким CTR — новый алгоритм повлиял на людей неравномерно.

### Результат бутстрепа
![Бутстреп](visualizations/ctr_bootstrap_distribution.png)

Два распределения не пересекаются — значит, разница между старым и новым алгоритмом реальна, а не случайна. Контрольная группа (синий) показывает лучший результат.
